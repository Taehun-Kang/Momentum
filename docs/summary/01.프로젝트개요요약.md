# 1. 프로젝트 개요 요약

> 상세 문서: [docs/basic/1.프로젝트 개요.md](../basic/1.프로젝트%20개요.md)

## 🎯 핵심 비전

"모든 순간에 딱 맞는 YouTube Shorts를 찾아주는 AI 큐레이터"

### 해결하는 문제

- **정보 과부하**: 하루 1억 5천만 개 업로드되는 Shorts 중 원하는 콘텐츠 찾기 어려움
- **컨텍스트 부족**: 기존 검색은 단순 키워드 매칭으로 상황별/감정별 검색 불가
- **개인화 한계**: YouTube 알고리즘은 과거 시청 기록에 의존, 현재 상황 반영 불가

### 기술적 차별화

- **MCP 기반 AI 큐레이션**: 65% → 90% 검색 정확도 향상
- **2단계 재생 가능성 필터링**: 임베드 불가능한 영상 사전 제거
- **실시간 트렌드 통합**: Bright Data MCP로 웹 전체 트렌드 수집
- **지능형 쿼리 빌더**: API 사용량 90% 절약 (100 units → 3 units)

## 👥 타겟 사용자 (3개 페르소나)

### 1. 트렌드 서퍼 (22세, 대학생)

- **이용 패턴**: 2-3시간/일, 최신 트렌드 중심
- **핵심 니즈**: "지금 뜨는 것", "친구들이 보는 것", "재미있는 것"
- **User Journey**:
  - 오전: 카페인서 "아침에 보기 좋은 영상"
  - 오후: 수업 중 "무음으로 볼 수 있는 영상"
  - 저녁: 친구들과 "같이 웃을 수 있는 영상"

### 2. 직장인 힐러 (31세, 개발자)

- **이용 패턴**: 30분-1시간/일, 힐링 콘텐츠
- **핵심 니즈**: "스트레스 해소", "마음의 평화", "짧은 휴식"
- **User Journey**:
  - 출근길: "지하철에서 볼 만한 영상"
  - 점심시간: "15분 브레이크에 딱 맞는 영상"
  - 퇴근 후: "하루 피로를 풀어주는 영상"

### 3. 콘텐츠 크리에이터 (27세, 유튜버)

- **이용 패턴**: 1-2시간/일, 리서치 목적
- **핵심 니즈**: "트렌드 분석", "경쟁사 모니터링", "영감"
- **User Journey**:
  - 아침: "어제 뜬 영상들" 분석
  - 오후: "내 채널과 비슷한 콘텐츠" 조사
  - 저녁: "새로운 아이디어" 발굴

## 🚀 핵심 기능 로드맵

### Phase 1: MVP (7일) - 기본 큐레이션

- ✅ **기본 검색**: 키워드 기반 Shorts 검색
- ✅ **사용자 인증**: Supabase Auth 기반
- ✅ **트렌드 표시**: 실시간 인기 키워드
- ✅ **재생 가능성 보장**: 2단계 필터링

### Phase 2: AI 고도화 (4일) - 대화형 검색

- 🔄 **AI 대화형 검색**: "지금 기분에 맞는 영상"
- 🔄 **컨텍스트 이해**: 시간, 장소, 감정 고려
- 🔄 **개인화 알고리즘**: 사용자 패턴 학습
- 🔄 **실시간 트렌드**: Bright Data MCP 연동

### Phase 3: 프리미엄 기능 (5일)

- 🔄 **고급 개인화**: ML 기반 추천
- 🔄 **크리에이터 도구**: 트렌드 분석 대시보드
- 🔄 **API 제공**: B2B 서비스
- 🔄 **오프라인 모드**: PWA 기반
