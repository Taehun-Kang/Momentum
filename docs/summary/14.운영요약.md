## 장애 대응

### 장애 등급
- **P0 (Critical)**: 전체 서비스 중단 → 5분 내 대응
- **P1 (Major)**: 핵심 기능 장애 → 15분 내 대응
- **P2 (Minor)**: 부가 기능 오류 → 1시간 내 대응

### 대응 절차
1. 장애 감지 → 등급 판단
2. 팀 알림 (Slack #incident)
3. 원인 파악 → 조치
4. 사용자 공지 (필요시)
5. 사후 분석 (RCA)
