## 🎯 주요 발견사항 요약

### 1. OR 연산의 치명적 문제점

```
"BTS | BTS 브이로그 | BTS 무대"
의도: 33% + 33% + 33%
실제: 36% + 4% + 60%
```

### 2. 단독 검색의 우수성

- BTS 댄스: 95% 키워드 매칭률
- BTS 브이로그: 85% 키워드 매칭률
- 속도: OR 연산 대비 2-3배 빠름

### 3. 2단계 필터링 필수성

- search.list (100 units) + videos.list (7 units)
- 재생 가능 여부 확인 절대 필요
- embeddable, privacyStatus, regionRestriction 체크
